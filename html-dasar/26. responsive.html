Note
  • Responsive dengan %, cuman bisa digunakan di element seperti gambar dan div
  • Pada kasus seperti gambar, mungkin ketika layar terlalu besar dan ukuran gambar kecil, secara otomatis gambar akan pecah, oleh karena itu biasanya pada kasus ini biasanya disediakan beberapa jenis ukuran gambar
  • Untuk melakukannya biasanya menggunakan CSS media query
  • Jika ingin melakukan perubahan responsive pada teks bisa menggunakan unit VW (Viewport Width)
