Note
  • Di semua tag html, terdapat atribut bernama ID
  • id merupakan attribut yang digunakan untuk menambahkan id ke element html yang kita buat
  • id harus bersifat unik disatu halaman html / tidak boleh sama
  • id akan sangan beguna jika kita menggunakan css dan javascript
  • dengan menggunakan id, kita bisa mengambil element menggunakan javascript lebih mudah
  • Dengan menggunakan ID, kita bisa mengubah gaya dengan CSS lebih mudah, apalagi jika terdapat tag yang sama. misal terdapat 5 tag div, kita hanya mengubah 1 div, kita bisa tambahkan id di div tersebut, dan menggunakan id untuk mengubah gaya pada div tersebut menggunakan css
